# 🌑 DarkCode

DarkCode is a web-based secret messaging platform that converts normal English text into a unique custom symbol language. Unlike Morse Code, DarkCode uses its own encoding system, allowing users to securely share and decode messages.

## ✨ Features

- Encrypt text into DarkCode
- Decrypt DarkCode back to text
- Real-time conversion
- Copy to clipboard
- Responsive dark-themed UI
- No login required

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/DarkCode.git
   ```

2. Open the project folder.

3. Run `index.html` in your browser.
